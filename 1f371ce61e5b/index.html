币安实验室领投 Plasm Network A 轮 240 万美元融资

美东时间2021年2月8日，币安（Binance）旗下的风险投资机构和孵化器币安实验室（Binance Labs）宣布领投波卡（Polkadot）网络上多虚拟机可拓展化智能合约平台 Plasm Network 的A轮融资，本轮融资总额达240万美元。

Plasm Network 是基于 Substrate 框架开发，支持包括 Optimistic 虚拟机（OVM）和 ZK-Rollups 等前沿 layer2 扩容解决方案，并同时兼容以太坊虚拟机（EVM）和 WASM，具有优越可拓展性及可互操作性的区块链基础设施。

Plasm Network 正式成为币安（Binance）用于支持波卡生态项目发展1000万美元基金的第一笔投资。这项投资将使 Plasm Network 加速网络研发、生态扩张和业务发展，并桥接币安生态和波卡生态。

“自上线以来，Plasm Network 的发展给我们留下了深刻的印象。作为第一条连接到波卡 Rococo V1 测试网的平行链，Plasm Network 同时也是波卡主网最具潜力的平行链竞争者之一。通过投资和支持 Plasm Network，我们展示了支持波卡生态成长和发展的承诺。” 币安实验室负责人周伟表示。

“我们非常高兴邀请币安加入 Plasm Network 生态。我们很高兴地知道，Plasm Network 是币安1000万美元波卡基金的第一笔投资。2021年对于 Plasm 和波卡来说是重要的一年。我们将尽最大努力使 Plasm 成为波卡上的 dApps 中心，并在币安实验室的支持下成为第一条 Kusama 和Polkadot 平行链。我们将一同开启 Web3.0 时代。” Plasm Network 创始人 Sota Watanabe 表示。

本轮融资是币安实验室对波卡生态系统的基础设施和创新项目赋能计划的一部分。参与本轮融资的还有 HashKey Capital、PAKA Ventures、Digital Finance Group 和 LongHash Ventures。

关于 Plasm Network

Plasm Network 致力于成为波卡（Polkadot）上领先的 dApps 中心，同时兼容以太坊虚拟机和WASM，并支持包括 OVM 和 ZK-Rollups 在内等前沿二层扩容方案。波卡中继链作为生态的核心，在设计上并不支持智能合约，使得波卡生态系统内的开发者亟需支持智能合约的中继链作为开发平台。凭借基于 dApps 生态的构建的经济模型，成为具有优越可拓展性、兼容性、可互操作性的智能合约平行链，正是 Plasm Network 的使命。最近，Plasm Network 成为首条接入波卡 Rococo V1 测试网的平行链，并推出了Plasm Network 的金丝雀网络 Shiden Network。我们计划成为第一条接入 Kusama（Shiden）和 Polkadot（Plasm）的平行链。

关于币安实验室（Binance Labs）

币安实验室是由币安（Binance）发起的区块链基础设施影响基金，旨在孵化、投资和赋能区块链和加密货币企业家、创新项目以及社区。币安实验室的使命是解决区块链生态系统的关键问题，让世界变得更好。

关于币安（Binance）

币安拥有全球领先的区块链资产交易平台，同时运营着整个币安生态体系，旗下包括币安交易所、币安实验室、币安Launchpad、币安智能链、币安资讯、币安学院、币安研究所、币安钱包和慈善基金。币安交易所是全球全球交易量最高的加密货币交易所，用户来自190多个国家和地区，每秒能够处理超过140万个订单，也是全球速度最快的交易所之一。币安平台的高安全性、稳定性和执行速度吸引大量区块链爱好者和专业交易者加入币安生态。
